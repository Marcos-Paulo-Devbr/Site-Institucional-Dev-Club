# DevClub — Site Institucional

Site institucional responsivo para apresentação do **DevClub**. O projeto foi construído com HTML, CSS e JavaScript , com foco em uma experiência visual moderna e interativa.

## Sobre o projeto

A página apresenta a proposta da escola, a trilha de formação, a equipe, dados da comunidade, empresas parceiras, perguntas frequentes e um convite para contato. Ela também reúne exemplos de projetos que os alunos podem desenvolver para o portfólio.

## Demonstração

Para visualizar o site, abra o arquivo `index.html` no navegador ou publique o repositório no GitHub Pages.

## Tecnologias utilizadas

- **HTML5** para a estrutura e semântica da página;
- **CSS3** para layout responsivo, animações e identidade visual;
- **JavaScript (Vanilla)** para as interações;
- **Canvas API** para a animação do fundo;
- **Intersection Observer API** para animações ao rolar a página;
- **Google Fonts** para as fontes Inter, Space Grotesk e JetBrains Mono.

## Funcionalidades

- Terminal animado na seção inicial;
- Contadores animados nas estatísticas;
- Carrosséis de módulos, projetos e empresas parceiras;
- Navegação por âncoras entre as seções;
- Efeitos de entrada durante a rolagem da página;
- Perguntas frequentes em formato de acordeão;
- Fundo em canvas com interação visual na navegação;
- Layout adaptado para desktop, tablet e celular;
- Respeito à preferência de redução de movimento do sistema;
- Páginas demonstrativas de projetos para portfólio.

## Seções da página

- Apresentação do DevClub e chamada principal;
- Estatísticas da formação;
- Missão, método e comunidade;
